Source for [maxanisimov.github.io](https://maxanisimov.github.io), my personal/academic site, built with [Jekyll](https://jekyllrb.com/) on the [academicpages](https://github.com/academicpages/academicpages.github.io) theme (itself based on [Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes/), © 2016 Michael Rose, MIT License — see LICENSE).

## Running locally

1. Install Ruby, Bundler, and Node.js
2. `bundle install`
3. `bundle exec jekyll serve` to build and serve at `localhost:4000`, rebuilding on change

Site content lives in `_pages/`, `_posts/`, `_publications/`, `_teaching/`; site-wide settings are in `_config.yml`; nav links are in `_data/navigation.yml`.
